📊 Monthly Expense Analyzer

A simple web-based expense tracking application built using HTML, CSS, and JavaScript.
This project allows users to enter their monthly expenses, analyze spending patterns using a Pie Chart, and download a detailed report as a Word document.

It helps users easily understand where their money is being spent and keep track of financial habits.



🚀 Features

📥 Add monthly expenses by category

📊 Visual expense analysis using Pie Chart

📄 Download expense report as a Word document (.doc)

🎨 Clean and responsive user interface

⚡ Fast and lightweight (No frameworks used)

📱 Works on desktop and mobile browsers




🛠️ Technologies Used

HTML5 – Structure of the application

CSS3 – Styling and responsive design

JavaScript (Vanilla JS) – Application logic

Chart.js – For displaying the Pie Chart analysis




📊 How It Works

User enters their monthly expenses with categories (Food, Travel, Bills, etc.).

The application stores and processes the data using JavaScript.

A Pie Chart visualization is generated to show spending distribution.

Users can download the monthly report as a Word document for records.
